# Roomora Interiors

Roomora Interiors is a premium luxury interior design and architecture portfolio website crafted to showcase timeless design excellence, modern spatial aesthetics, and high-end project presentation.

Designed with a refined visual identity, the platform delivers an immersive digital experience that reflects sophistication, creativity, and architectural precision. Every section is thoughtfully structured to present services, showcase curated projects, highlight design expertise, and create seamless engagement for potential clients.

The website serves as a complete digital presence for a luxury interior design studio, combining cinematic visuals, elegant transitions, and interactive storytelling to represent the brand’s philosophy of transforming spaces into timeless masterpieces.

---

## Overview

Roomora Interiors is built as a modern multi-page web experience focused on luxury branding and portfolio presentation.

The platform allows visitors to:

* Explore premium residential, commercial, and bespoke interior projects
* Learn about the studio’s design philosophy
* Discover offered services and consultation packages
* Browse detailed project case studies
* Submit inquiries directly through interactive contact forms
* Subscribe to studio updates and design inspiration

Every page is optimized for visual storytelling and crafted to create a polished first impression.

## Technologies Used

### Frontend Framework

**React.js**

Provides component-based architecture for scalable and maintainable development.

---

### Routing

**React Router**

Used for dynamic page navigation and project detail routing.

---

### Animation

**Framer Motion**

Powers smooth page transitions, reveal animations, hover effects, and motion interactions throughout the experience.

---

### Form Management

**React Hook Form**

Handles form state efficiently for inquiry and newsletter submission.

---

### HTTP Requests

**Axios**

Used for backend communication including contact submissions and newsletter requests.

---

### Notifications

**React Toastify**

Provides polished real-time user feedback for actions like submissions and subscriptions.

---

### Icons

**React Icons**

Used for elegant and scalable iconography.

---

### Styling

**Tailwind CSS**

Responsible for the entire design system including:

* Layout
* Typography
* Color system
* Responsive behavior
* Animations
* Utility-driven styling
